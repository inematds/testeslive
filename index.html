Index do Projeto

O que e este projeto?

Este repositorio reune uma colecao de skills para Claude, com exemplos praticos de instrucoes, scripts e recursos para tarefas especializadas.

A ideia principal e mostrar como organizar skills de forma reutilizavel, com foco em produtividade, padrao de qualidade e uso no mundo real.

Estrutura principal

skills/

Exemplos prontos de skills para diferentes casos (design, tecnico, docs, etc).

spec/

Especificacao do formato de Agent Skills.

template/

Template base para criar novas skills com SKILL.md.

Como usar

  • Instalar skills no Claude Code via plugin marketplace.
  • Usar uma skill mencionando seu nome durante a tarefa.
  • Criar skill propria com base no template e no arquivo README.md.

Resumo rapido

Em poucas palavras: este projeto e um catalogo pratico para aprender, adaptar e publicar skills que ampliam as capacidades do agente.
